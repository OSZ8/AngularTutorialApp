Angular Tutorial

Bienvenido al tutorial de Angular. Aprende cómo usar este framework paso a paso.

1. Justificación del Framework

Angular es un framework de desarrollo web basado en TypeScript que permite crear aplicaciones dinámicas, escalables y de alto rendimiento.

Se utiliza principalmente para desarrollar Single Page Applications (SPA), donde el contenido se actualiza dinámicamente sin necesidad de recargar la página.

  • ¿Para qué sirve? Angular se usa para construir aplicaciones web robustas con funcionalidades complejas y una experiencia de usuario fluida.
  • ¿Qué aplicaciones tiene? Angular es ideal para aplicaciones empresariales, paneles de control, aplicaciones móviles híbridas y plataformas de comercio electrónico.
  • ¿Sobre qué lenguajes se apoya? Está basado en TypeScript, un superset de JavaScript que incluye tipado estático.
  • ¿Qué prerrequisitos necesita? Para usar Angular necesitas conocimientos básicos de HTML, CSS y JavaScript.

2. Instalación

Para comenzar a usar Angular, necesitas seguir estos pasos:

  1. Instalar Node.js: Descarga e instala Node.js desde su página oficial: https://nodejs.org. Esto instalará npm (Node Package Manager) automáticamente.
  2. Instalar Angular CLI: Una vez tengas npm, instala Angular CLI con el siguiente comando en tu terminal: npm install -g @angular/cli.
  3. Crear un nuevo proyecto: Crea tu primer proyecto Angular ejecutando: ng new nombre-del-proyecto.
  4. Ejecutar el servidor de desarrollo: Navega al directorio del proyecto con: cd nombre-del-proyecto, y luego ejecuta: ng serve. Abre tu navegador en http://localhost:4200 para ver la aplicación funcionando.

3. Primeros Pasos

Una vez que hayas creado tu proyecto Angular, puedes comenzar modificando el componente principal para ver cómo funciona. Aquí tienes un ejemplo para mostrar un "Hola Mundo":

  1. Abre el archivo src/app/app.component.ts en tu editor de código.
  2. Sustituye el contenido del archivo por el siguiente código: 
            
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  template: `
    <h1>Hola Mundo desde Angular!</h1>
  `,
  styles: [`
    h1 {
      color: green;
      text-align: center;
    }
  `]
})
export class AppComponent {}
  3. Guarda los cambios y recarga la página. Deberías ver el mensaje "Hola Mundo desde Angular!" en tu navegador.

4. Utilización del Framework

Angular se basa en una arquitectura de componentes que permite dividir la aplicación en partes reutilizables y fácilmente gestionables. A continuación, te mostramos los conceptos clave:

  • Componentes: Son las piezas fundamentales de cualquier aplicación Angular. Cada componente consta de un archivo HTML (vista), CSS (estilos) y TypeScript (lógica).
  • Módulos: Los módulos agrupan componentes relacionados y gestionan la organización de la aplicación. El módulo principal es AppModule.
  • Servicios: Los servicios permiten compartir lógica o datos entre varios componentes. Se definen con decoradores como @Injectable.
  • Enrutamiento: Angular incluye un sistema de rutas que permite navegar entre diferentes páginas o vistas dentro de la aplicación.

A continuación, un ejemplo práctico para añadir un nuevo componente:

  1. Crea un nuevo componente ejecutando el siguiente comando en tu terminal: ng generate component nombre-del-componente.
  2. Angular generará automáticamente los archivos necesarios: nombre-del-componente.component.ts, .html, .css, y .spec.ts.
  3. Edita el archivo nombre-del-componente.component.html para agregar contenido, por ejemplo: 
            
<h2>Bienvenido al nuevo componente!</h2>
  4. Incluye el componente en la vista principal usando su selector, por ejemplo: <app-nombre-del-componente></app-nombre-del-componente>.

5. Explicación del Funcionamiento del Framework

Angular organiza una aplicación en módulos y componentes que trabajan juntos para manejar la lógica, la presentación y la navegación. Aquí te explicamos cómo funciona:

  • Páginas y vistas: Cada página en Angular está representada por un componente, que controla su contenido (HTML), estilos (CSS) y comportamiento (TypeScript).
  • Ficheros clave: Cada componente consta de:
    • .component.ts: Define la lógica y el comportamiento del componente.
    • .component.html: Define la estructura visual de la página.
    • .component.css: Contiene los estilos específicos del componente.
  • Interacción entre componentes: Los componentes pueden comunicarse mediante entradas (@Input) y salidas (@Output), lo que permite compartir datos.
  • Enrutamiento: Angular utiliza un sistema de rutas definido en app-routing.module.ts para conectar diferentes componentes con las URLs correspondientes.

Por ejemplo, para crear una nueva página:

  1. Crea un componente usando: ng generate component nombre-del-componente.
  2. Define su ruta en el archivo app-routing.module.ts, así: 
            
const routes: Routes = [
  { path: 'nuevo', component: NuevoComponenteComponent }
];
  3. Navega a /nuevo en tu navegador para ver la nueva página.

6. Conclusiones

Angular es un framework poderoso y versátil para desarrollar aplicaciones web modernas. Su arquitectura basada en componentes y módulos permite crear proyectos escalables y fácilmente mantenibles.

A lo largo de este tutorial, hemos explorado:

  • La importancia y justificación de usar Angular.
  • Los pasos necesarios para instalar y configurar un proyecto.
  • Cómo realizar una primera aproximación con un ejemplo básico.
  • Los conceptos clave para estructurar y expandir aplicaciones, como componentes, servicios y enrutamiento.

Angular tiene una curva de aprendizaje moderada, pero su ecosistema robusto y la comunidad activa ofrecen muchos recursos para superar cualquier desafío.
